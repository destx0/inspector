inspenctor

On-screen inspection tools for Angular. Measure spacing, inspect typography, place guides.

Install

npm install inspenctor

Usage

<inspenctor-overlay
  [persistOnReload]="true"
  [hoverHighlightEnabled]="true"
/>

Shortcuts

MToggle inspector
SSelect mode
GGuides mode
H / VGuide orientation
AltMeasure spacing
EscClear

Test Canvas

Header
Meta

Product

Measure, inspect, and align UI without leaving the browser.

Typography mode annotates visible text blocks. Selection mode shows font size, line height, hex color, padding, margin, and gap context.

Typography

Check font size, line height, and color labels.

Spacing

Hold Alt to measure gap to another section.

Guides

Place persistent guides from toolbar.
