Angular 17 Library

inspenctor

Compact on-screen inspection tools for Angular apps. Measure spacing, inspect typography, place guides, and check layout relationships directly in the browser.

Installation Usage
npm install inspenctor
  • Select elements and inspect typography
  • Annotate text blocks with font size, line height, and hex color
  • Hold Alt to measure section spacing
  • Create horizontal and vertical guides

Installation

Install the package

Add the library to your Angular app, then render the overlay once near the root of your application. 

npm install inspenctor
pnpm add inspenctor

Usage

Mount the overlay once

Keep your app content unchanged and place the inspector at the end of the root template so it can observe the whole page. 

<app-shell />

<inspenctor-overlay
  [persistOnReload]="true"
  [hoverHighlightEnabled]="true"
/>

Shortcuts

Default controls

MToggle inspector
SSelect mode
GGuides mode
H / VGuide orientation
AltMeasure spacing between sections
EscClear active state
